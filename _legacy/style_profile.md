# 默认风格画像（CN121526509A）

## 来源
`writing-style-analyzer` 对 CN121526509A 的风格提取，作为默认写作规范。

## 核心风格规则

### 固定起手句
实施例开头统一使用：`本发明实施例提供一种...方法，所述方法包括：`

### 步骤总述链
以 S101、S102... 编号串联方法步骤，每个步骤包含动作描述、子步骤、技术效果。

### 解释型说明书笔法
对每个步骤说明：做什么 → 怎么做 → 为什么这样做 → 达到什么效果。

### 长句链
步骤描述使用长句，以分号分隔多个子动作，保持语义连贯。

### 对偶归纳句
每段实施例末尾以对偶句收尾，概括技术效果和进步。

## 脱敏规则
仅在 Phase 1 确认有本地项目材料时启用：
- 公司/产品名 → "某系统"
- 客户/项目标签 → "对象A"  
- 内部路径 → "本地路径"
- 真实分类标签 → "分类A"
- 具体业务规模 → "一定规模"
